미로 찾기 게임 설명

게임 목표: 말(플레이어)을 시작 지점(⛩️)에서 도착 지점(🚩)까지 이동시켜 미로를 탈출하세요.

조작법:
- 키보드 방식: 키보드의 화살표 키(⬆️⬇️⬅️➡️)를 사용하세요.
- 조이스틱 방식: 화면의 조이스틱을 드래그하여 이동합니다.
- Q 버튼 (시작위치): 시작 지점으로 즉시 이동합니다. (게임당 1회)
- W 버튼 (동선): 지나온 길을 표시합니다. (게임당 1회)

특수 기능:
- 롤백 (1, 2): 현재 위치를 저장하거나, 저장된 위치로 이동합니다.

게임 관리:
- 크기 설정: 시작 화면으로 돌아가 새로운 미로 크기를 설정합니다.
- 다시 시작: 현재 크기로 새로운 미로를 생성하여 재시작합니다.
